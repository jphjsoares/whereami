How does this work?

  • To start a game you need a map id. Or you can also try a random map.

  • If you don't have a map id, please create a map by clicking "Create a map".

  • After you create a map, a popup box will be shown with the id of the map you just created.

  • When you have a valid map id either created by you or given to you by a friend, click "I have a map". Paste the id on there and you are good to go!

  • At the end of each game you can share the link with the game summary and share it with your friends.

  • Good luck!
